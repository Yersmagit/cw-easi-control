## 介绍

EasiControl 是一个 ClassWidgets 插件，可以方便地实现 ClassWidgets 的自动化控制。


### 截图
![截图1](none)

### 特性

#### 已实现：
- 配合插件 [LX-Music-Lyrics-Plugin](https://github.com/laoshuikaixue/cw-LX-music-lyrics-plugin) 自动改变您的 ClassWidgets 小组件类型
- 特定活动时显示特定的小组件
- ...

#### 待实现：
- 高度自定义执行特定事件的条件
- 高度自定义特定条件下执行的事件
- ...
